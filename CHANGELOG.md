### Added
- Initial platform release
- `apex_token` Soroban contract — fungible APEX reward token
- `apex_campaign` Soroban contract — merchant campaign management
- Express REST API with JWT authentication
- Next.js 14 frontend with Freighter wallet integration
- Docker Compose full-stack development environment
- Kubernetes manifests and Helm chart
- Terraform infrastructure modules (VPC, RDS, EC2, ElastiCache)
- Prometheus + Grafana monitoring stack
- GitHub Actions CI/CD pipelines
