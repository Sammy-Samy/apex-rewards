# Apex Rewards Roadmap

## v1.0 — Foundation (current)
- [x] Soroban smart contracts: `apex_token`, `apex_campaign`
- [x] Express REST API with JWT auth
- [x] Next.js 14 frontend with Freighter wallet
- [x] Docker Compose dev environment
- [x] CI/CD with GitHub Actions

## v1.1 — Merchant Portal
- [ ] Merchant dashboard with campaign analytics
- [ ] Bulk reward issuance
- [ ] Campaign expiry and auto-close

## v1.2 — Customer Experience
- [ ] Mobile-responsive PWA
- [ ] Push notifications for earned rewards
- [ ] QR code redemption flow

## v2.0 — Ecosystem
- [ ] Multi-token support (USDC redemptions)
- [ ] Cross-merchant reward swaps
- [ ] On-chain reputation scoring
- [ ] Stellar DEX integration for APEX liquidity
